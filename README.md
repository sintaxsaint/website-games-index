# Website Games Index

This project hosts a collection of games accessible via a central loader. It is hosted on GitHub Pages.

## About
This project is maintained by a single developer (**sintaxsaint**) helped by some A.I. so updates and new game additions may take some time.

## How to Play

1.  Browse the list of available games on the [Main Page](index.html).
2.  Click on a game to start playing.
3.  You can also request new games via the [Request Page](request.html).

## Usage Examples

### Direct Link
To link to a specific game, use the ID provided in the game list:
`https://sintaxsaint.github.io/website-games-index/call.html?id=<GAME_ID>`

### Iframe Embedding
You can embed games on other websites using an iframe:

```html
<iframe 
    src="https://sintaxsaint.github.io/website-games-index/call.html?id=1" 
    width="800" 
    height="600" 
    frameborder="0" 
    allowfullscreen>
</iframe>
```
